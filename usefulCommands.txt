Rodar o backend com o banco zerado
docker compose exec backend python manage.py flush

Criar novo app
docker compose exec backend python manage.py startapp <appName>

Rodar serviço específico
docker compose up <service>

Ver quais serviços estão rodando
docker compose ps

Acessar o banco diretamente para rodar consultas (exit com \q)
docker compose exec db psql -U appuser -d appdb

Fazer migrações
python manage.py makemigrations (dentro de backend/backend)
docker compose exec backend python manage.py makemigrations
docker compose exec backend python manage.py migrate

Deletar Migrações:
Dentro da pasta backend:
Get-ChildItem -Recurse -Include *.py -Path *\migrations | Where-Object { $_.Name -ne "__init__.py" } | Remove-Item
Get-ChildItem -Recurse -Include *.pyc -Path *\migrations | Remove-Item

Resetar banco
docker compose down -v
docker compose up --build

Refazer migrações:
docker compose exec backend python manage.py makemigrations
docker compose exec backend python manage.py migrate

type init_admin.py | python manage.py shell

Get-ChildItem -Recurse -Filter "__pycache__" | Remove-Item -Recurse -Force
